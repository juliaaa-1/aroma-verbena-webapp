- `assets/image/maria.jpg` - фото Марии (вертикальное, рекомендуется 400x600px)
- `assets/image/maria-circle.jpg` - круглое фото Марии (квадратное, 300x300px)
- `assets/image/интерьер/interior1.jpg` - фото интерьера (минимум 4 фото)
- `assets/image/интерьер/interior2.jpg`
- `assets/image/интерьер/interior3.jpg`
- `assets/image/интерьер/interior4.jpg`
- `assets/image/услуги/individual.jpg` - для услуги "Индивидуальная ароматерапия"
- `assets/image/услуги/personal.jpg` - для услуги "Персональный подбор масел"
- `assets/image/услуги/massage.jpg` - для услуги "Массажи и телесные практики"
- `assets/image/услуги/education.jpg` - для услуги "Обучение и мастер-классы"
- `assets/image/услуги/diagnostic.jpg` - для услуги "Аромадиагностика"

- `assets/image/отзывы/avatar1.jpg` - аватарки для отзывов (круглые, 120x120px)
- `assets/image/отзывы/avatar2.jpg`
- `assets/image/отзывы/avatar3.jpg`

### 2. Добавить шрифты (опционально)

Если хотите использовать свои шрифты:
1. Поместите файлы шрифтов в `assets/fonts/`
2. Добавьте `@font-face` правила в `assets/style/style.css`

### 3. Настроить Яндекс карту

1. Получите API ключ на https://developer.tech.yandex.ru/
2. Откройте `index.html`
3. Найдите строку с Яндекс картой (в конце файла)
4. Замените `YOUR_API_KEY` на ваш реальный API ключ

### 4. Обновить отзывы

В файле `index.html` найдите секцию `.reviews-track` и замените тестовые отзывы на реальные:
- Измените тексты отзывов
- Измените имена
- Добавьте больше отзывов при необходимости

### 5. Обновить баннер (опционально)

Если хотите добавить фоновое изображение для баннера:
1. Добавьте изображение в `assets/image/banner-bg.jpg`
2. В `assets/style/style.css` найдите `.banner` и добавьте:
```css
.banner {
    background-image: url('../image/banner-bg.jpg');
    background-size: cover;
    background-position: center;
}
```

## Как запустить:

### Локально (для просмотра):

1. Откройте `index.html` в браузере
2. Или используйте локальный сервер:
   ```bash
   # Python 3
   python -m http.server 8000
   
   # Или через Node.js (если установлен)
   npx http-server
   ```
3. Откройте в браузере `http://localhost:8000`

### Для Telegram бота:

Если у вас уже настроен сервер для webapp, просто замените файлы:
- `index.html`
- `script.js`
- `assets/style/style.css`

И добавьте новые изображения в папку `assets/image/`
